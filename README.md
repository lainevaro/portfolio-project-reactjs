# Alvaro Portfolio ReactJS + Vite
## How to use

Run the next command to install the dependencies:

`yarn install`

then run:

`yarn dev`

to start the development server.

You can fin other scripts in the `package.json` file:

```
"scripts": { 
    "dev": "vite", 
    "build": "tsc && vite build", 
    "preview": "vite preview" 
},
